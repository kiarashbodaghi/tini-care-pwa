# TINI‑Care (PWA) — Ultimate v0.4
وب‌اپ مدیریت علائم وزوز گوش با تمرکز بر تمرین‌های صوتی (تون/نویز/نُوچ/AM)، آزمون RI، ارزیابی THI/TFI و نمودار پیشرفت.

## دمو
- نسخه GitHub Pages (پس از آپلود): https://<username>.github.io/tini-care-pwa/

## ویژگی‌ها
- Web Audio API (تون، نویز سفید/صورتی/قهوه‌ای، فیلتر نُوچ، AM Noise)
- آزمون RI (۶۰ ثانیه) با ثبت امتیاز و یادداشت
- THI/TFI کوتاه (۱۰ سوال)
- نمودارهای Canvas برای پیشرفت
- PWA کامل (manifest, service worker با آپدیت سریع)
- داده‌ها فقط در دستگاه کاربر (localStorage)
